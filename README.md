### Installation

Clone the repository.

Update mongodb connection string in config/config.env

Install the dependencies and devDependencies and start the server.

```sh
$ cd mern-shopping-cart
$ npm install
$ cd client
$ npm install
```

Run the server

```sh
$ cd mern-shopping-cart
$ npm run server
```

Run the client
```sh
$ cd mern-shopping-cart
$ cd client
$ npm run start
```
